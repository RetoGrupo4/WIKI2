INSTALACION

Como instalar Prestashop en el servidor

-Primero crearemos la base de datos para el prestashop en mysql

  • Entramos en el mysql

    • mysql -u root -p

  • Creamos la base de datos

    • create database prestashop

  • Le asignamos un password y permisos de control total sobre la base de datos.

    • grant all PRIVILEGES on prestashop.* to prestashop@localhost identified by 'password';

  • Hay que asegurarse siempre de refrescar todos los privilegios.

    • FLUSH PRIVILEGES;

-Iniciamos la instalacion descargando el paquete de prestashop

  • wget http://www.prestashop.com/download/releases/prestashop_1.7.4.3.zip

-Descomprimimos el paquete descargado

  • unzip prestashop_1.7.4.3.zip

-Descomprimimos un nuevo paquete que aparece cuando extraemos el descargado

  • unzip prestashop.zip

-Damos los permisos a la carpeta descomprimida y cambiamos de dueño

  • chown -R www-data:www-data owncloud
  • chmod 755 owncloud

-Ponemos nuestra ip asignada en el servidor y la carpeta prestashop para abrir el index.php y nos dara fallo asique tenemos que crear un archivo de configuracion.

  • 10.9.52.107/prestashop
  • nano /etc/apache2/sites-available/prestashop.conf

-Después de configurar el VirtualHost anterior, habilítelo ejecutando los comandos a continuación . Por ultimo reiniciamos el apache2

  • a2ensite prestashop.conf
  • a2enmod rewrite
  • service apache restart

-Volvemos a entrar y ahora nos permite empezar a instalar.

-Siguiendo los pasos de instalacion no nos deja continuar porque tenemos que instalar unos paquetes del php.

  • sudo apt install php libapache2-mod-php php-common php-mbstring php-xmlrpc php-soap php-gd php-xml php-intl php-mysql php-cli php-mcrypt php-ldap php-zip php-curl

-Instalado el php que faltaba ya nos deja continuar y a continuación tendremos que rellanar los siguientes campos.

-Ahora tendremos que rellanar los siguientes campos con los datos de la base de datos.

-Esperamos que finalice la instalacion y una vez terminada ya podras acceder a tu prestashop .

-Antes de entrar tendremos que borrar el archivo install dentro de la carpeta prestashop y una vez borrado solo tendriamos que entrar por admin y iniciar sesion.

  • rm -r install
  • 10.9.52.107/prestashop/admin038vmsoyg
