WORDPRESS

Que es?

Sistema de gestión de contenidos enfocado a la creación de cualquier tipo de página web.

Está desarrollado en el lenguaje PHP para entornos que ejecuten MySQL y Apache, bajo licencia GPL y es software libre.

Aunque se basa en la utilización del motor de bases de datos MySQL sobre el servidor web Apache, éste sistema de gestión de contenidos puede también instalarse en entornos que ejecuten Postgresql, MariaDB y Nginx, Pueden verse pros y contras de la utilización de bases de datos alternativas en la documentación oficial de Codex. El mismo sitio puede ser consultado a la hora de planificar una instalación basada en Nginx.
