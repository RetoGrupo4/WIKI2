INSTALACION

Como hemos instalado Owncloud en el servidor

  • Entramos en el mysql

    • mysql -u root -p

  • Creamos la base de datos

    • create database owncloud

  • Le asignamos un password y permisos de control total sobre la base de datos.

    • grant all PRIVILEGES on owncloud.* to owncloud@localhost identified by 'password';

  • Hay que asegurarse siempre de refrescar todos los privilegios.

    • FLUSH PRIVILEGES;

-Iniciamos la instalacion descargando el paquete de owncloud

  • wget http://download.owncloud.org/community/owncloud-10.0.10.zip

-Descomprimimos el paquete descargado

  • unzip owncloud-10.0.10.zip

-Damos los permisos a la carpeta descomprimida y cambiamos de due√±o

  • chown -R www-data:www-data owncloud
  • chmod 755 owncloud

-Ponemos nuestra ip asignada en el servidor y la carpeta owncloud para abrir el index.php y nos mostrara los datos que necesita para la instalacion.

  • 10.9.52.107/owncloud

-Rellenamos los datos de la base de datos y pulsamos submit.

-Iniciamos sesion.

-Y ya estaria instalado wordpress en nuestro servidor.
