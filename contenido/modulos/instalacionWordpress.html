Como instalar Wordpress en el servidor

Primero crearemos la base de datos para el wordpress en mysql

  • Entramos en el mysql

    • mysql -u root -p

  • Creamos la base de datos

    • create database wiki2

  • Le asignamos un password y permisos de control total sobre la base de datos.

    • grant all PRIVILEGES on wiki2.* to wordpress@localhost identified by 'password';

  • Hay que asegurarse siempre de refrescar todos los privilegios.

    • FLUSH PRIVILEGES;

-Iniciamos la instalacion descargando el paquete de wordpress

  • wget http://wordpress.org/latest.tar.gz

-Descomprimimos el paquete descargado

  • tar -xzvf latest.tar.gz

-Damos los permisos a la carpeta descomprimida

  • chmod 777 wordpress

-Ponemos nuestra ip asignada en el servidor y la carpeta wordpress para abrir el index.php y nos mostrara los datos que necesita para la instalacion pulsamos LETÂ´S GO.

  • 10.9.52.107/wordpress

-Rellenamos los datos de la base de datos y pulsamos submit.

-Iniciamos la instalacion de wordpress rellenamos los campos con los siguientes datos.

-Iniciamos sesion.

-Y ya estaria instalado wordpress en nuestro servidor.
